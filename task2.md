# 미니게임 개선 작업 2

## 목표

발사 각도와 힘이 실제 궤적에 분명하게 반영되도록 물리를 안정화하고, 아이템을 직접 밟아 극적인 효과를 체감할 수 있는 속도와 화면 구성을 만든다.

## 구현 항목

- [x] 차량 바닥을 도로 상단에 맞추고 차량 표시 크기를 기존의 70%로 축소한다.
- [x] 발사 방향 화살표를 기존 길이의 2배로 늘린다.
- [x] 비정상적인 초기 속도 배수를 제거하고 아이템 충돌이 가능한 속도 범위로 제한한다.
- [x] 선택한 각도와 힘이 수평·수직 속도, 체공 시간, 바운드에 계속 영향을 주도록 한다.
- [x] 점프 및 에너지 아이템 효과를 기준 효과의 25배 수준으로 강화하되 수평 속도에는 플레이 가능한 상한을 둔다.
- [x] 강한 힘과 위쪽 각도에서는 차량이 화면 위로 사라지게 한다.
- [x] 화면 밖 차량의 방향을 가리키는 큰 화살표와 현재 고도(m)를 상단에 표시한다.
- [x] 각도·힘 설정 패널을 아이템 범례보다 위에 배치해 겹치지 않게 한다.

## 완료 기준

1. 최대 힘 발사라도 첫 아이템을 건너뛰지 않고 충돌 판정이 안정적으로 동작한다.
2. 낮은 각도와 높은 각도의 궤적 차이가 눈에 띈다.
3. 점프 발판은 차량을 화면 밖까지 올리고, 에너지 아이템은 즉시 뚜렷한 가속을 만든다.
4. 차량이 화면 밖에 있을 때 화살표와 미터 단위 고도가 보이며, 복귀하면 표시가 사라진다.
5. 차량 이미지의 최하단이 도로 상단에 닿아 떠 보이지 않는다.
