# 자동차 멀리 날리기 미니게임 수정 작업

## 목표

자동차를 항상 화면에 포커싱하고, 체감 속도와 자동차/아이템 UI를 정상화한다.

## 요구사항 및 완료 기준

- [x] 자동차 이미지가 왼쪽을 향하면 좌우 반전하여 헤드램프가 진행 방향(오른쪽)을 보게 한다.
- [x] 자동차 바닥이 도로 지면선에 정확히 닿도록 배치한다.
- [x] 자동차 표시 크기를 기존의 4배로 확대한다.
- [x] 각도/파워 설정창 너비를 기존의 3/5 수준으로 줄인다.
- [x] 각도를 정한 뒤 버튼을 누르는 동안 파워 게이지가 왕복하고, 버튼을 떼는 순간 파워를 확정한다.
- [x] 비행 중 카메라가 자동차를 화면 안의 고정 포커스 지점에 유지한다.
- [x] 발사 속도를 현재 값의 1/50로 줄인다.
- [x] AI로 수정된 자동차 이미지가 선택 데이터에 저장된 경우 미니게임도 그 이미지를 그대로 사용한다.
- [x] 아이템의 게임 내 표시를 범례와 같은 이미지(연료/배터리, 점프 발판, 가시/나무, 벌꿀)로 통일한다.
- [x] 좌측 하단 아이템 설명을 한 줄 카드형 UI로 키우고, 자동차와 겹치거나 게임 조작을 방해하지 않게 한다.

## 우선순위

1. 자동차 카메라 포커싱
2. 아이템 표시 및 설명 UI
3. 속도 1/50 감속
4. 자동차 방향, 접지, 크기
5. 각도/파워 UI
