# 퇴직연금 계산 도구 시나리오

## 1. 왜 만드는가

효율적인 퇴직연금 계산을 위해 만든다.

퇴직연금 담당자가 반복적으로 수행하는 계산과 기록 관리를 한 곳에서 처리할 수 있게 하여 업무 시간을 줄이고, 인원별 정보를 쉽게 확인할 수 있도록 한다.

## 2. 누가 사용하는가

퇴직연금 담당자 1명이 사용한다.

여러 사용자의 동시 사용이나 권한 관리는 고려하지 않는다.

## 3. 어떤 상황에서 쓰는가

PC 환경에서 사용한다.

담당자가 업무 중 직원별 퇴직연금 관련 정보를 확인하거나 새로 작성하고, 필요 없는 데이터를 삭제할 때 사용한다.

## 4. 핵심 기능 목록

- 인원별 목록 보기
- 퇴직연금 정보 글쓰기
- 퇴직연금 정보 삭제

## 5. 핵심 목표

퇴직연금 담당자가 PC에서 인원별 퇴직연금 정보를 목록으로 확인할 수 있다.

담당자가 필요한 퇴직연금 정보를 새로 작성하고 저장할 수 있다.

담당자가 더 이상 필요 없는 퇴직연금 정보를 삭제할 수 있으면 완료된 상태로 본다.
